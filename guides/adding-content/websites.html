Adding from Websites ​

Adding content from websites.

Importing a series ​

  1. Go to the Plugins page and install a plugin. Plugins add support for multiple content sources.
  2. Go to the Add Series page and select a source from the dropdown. Search for a series to add to your library.

You can view the series you added on the Library page.

Settings ​

Plugins can have settings, including ones for individual content sources. These can be configured by clicking Settings on the Plugins page after installing a plugin.
